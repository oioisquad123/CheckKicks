# Task ID: 21.5

**Title:** Create app icon and generate all required asset sizes

**Status:** pending

**Dependencies:** None

**Priority:** medium

**Phase:** 1D - Pre-Monetization Testing

**Description:** Design 1024x1024 app icon reflecting CheckKicks brand and generate all required iOS app icon sizes for TestFlight and App Store submission

**PRD Design Guidelines:**
- **App Name:** CheckKicks
- **Theme:** Modern, trustworthy, tech-forward
- **Primary Color:** Deep navy #1E3A5F or black (premium feel)
- **Accent Color:** Green #22C55E (authentic)
- **Typography:** SF Pro (system)

**Details:**

**1. Design Concept**
Choose one of these approaches:

**Option A: Badge/Shield Design**
- Circular or shield badge shape
- Checkmark or verification symbol
- "A" lettermark for CheckKicks
- Green accent on deep navy background
- Conveys: Trust, verification, authority

**Option B: Tech/AI Design**
- Abstract sneaker outline
- AI circuit pattern or neural network
- Modern geometric shapes
- Gradient from navy to green
- Conveys: Technology, AI-powered, modern

**Option C: Minimalist "A" Lettermark**
- Bold "A" letterform
- Integrated checkmark or verification tick
- Solid background (navy or black)
- Simple, clean, memorable
- Conveys: Simplicity, authenticity, direct

**Recommended:** Option C (Minimalist "A") - Most recognizable at small sizes

**2. Design Requirements**

**Technical Specs:**
- Size: 1024x1024 pixels
- Format: PNG (no transparency for iOS)
- Color space: sRGB or Display P3
- No alpha channel
- No rounded corners (iOS adds automatically)
- Safe area: Keep important content within 80% center

**Design Guidelines:**
- Avoid text (too small at icon sizes)
- Use simple, bold shapes
- High contrast for visibility
- Test at small sizes (60x60, 40x40)
- Avoid fine details that disappear when scaled
- No iOS UI elements (no literal screenshots)
- No Apple hardware images
- Distinct from competitors

**4. Generate All Required Sizes**

Use one of these tools to generate all iOS app icon sizes from your 1024x1024 master:

**Recommended Tool: Icon Set Creator (Free)**
- Website: https://www.appicon.co
- Upload 1024x1024 PNG
- Download iOS asset catalog

**Alternative: Manual Xcode**
- Drag 1024x1024 into Xcode Assets.xcassets → AppIcon
- Xcode generates other sizes automatically (iOS 11+)

**Required Sizes (iOS):**
- 1024x1024 (App Store)
- 180x180 (iPhone, @3x)
- 120x120 (iPhone, @2x)
- 87x87 (iPhone, Settings @3x)
- 80x80 (iPhone, Spotlight @2x)
- 60x60 (iPhone, Notification @3x)
- 58x58 (iPhone, Settings @2x)
- 40x40 (iPhone, Spotlight @2x)

**5. Add to Xcode Project**

**Steps:**
1. Open `CheckKicks_AI.xcodeproj` in Xcode
2. Navigate to Assets.xcassets
3. Select AppIcon
4. Drag 1024x1024 PNG into "App Store iOS 1024pt" slot
5. If using auto-generation, Xcode fills other slots
6. If using Icon Set Creator, drag all sizes into respective slots
7. Verify no warnings in AppIcon set
